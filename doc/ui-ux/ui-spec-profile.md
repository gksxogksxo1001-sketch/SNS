# [UI Spec] 프로필 및 여행 아카이브

## 1. 개요
나의 여행 정체성을 표현하고, 지금까지의 여행 기록을 지도와 통계로 한눈에 확인하는 개인 공간입니다.

## 2. 화면 구성 요소

### 상단 영역 (Profile Header)
- **프로필 이미지**: 원형 프레임
- **닉네임 및 소개**: 닉네임 아래에 한 줄 소개글
- **팔로워/팔로잉 수**: 클릭 시 목록으로 이동
- **[프로필 편집]** 버튼

### 하이라이트 영역 (Travel Info)
- **여행 스타일 칩**: 사용자가 설정한 태그들 (예: #배낭여행 #가성비)
- **여행 통계 바**: [방문 도시 12] | [총 거리 2,450km] | [정산 완료 95%]

### 메인 콘텐츠 영역 (Archive)
- **선택 탭**: [지도 보기(기본)] | [그리드 보기(게시물)]
- **지도 보기**: 
    - 전 세계/전국 지도가 나타남
    - 게시물을 올린 위치에 마커(핀) 표시
    - 핀 클릭 시 해당 여행 게시물 요약 카드 팝업
- **그리드 보기**: 인스타그램 스타일의 3열 사진 배열

## 3. 주요 인터랙션
- **지도 줌 인/아웃**: 특정 지역의 여행 핀들을 모아보기
- **탭 전환**: 지도와 사진 리스트 간의 부드러운 전환 애니메이션
- **통계 칩 클릭**: 상세 여행 리포트 페이지(추후 확장)로 이동 가능성 고려

## 4. 디자인 포인트
- 개인의 성취감을 높여줄 수 있는 '누적 데이터' 시각화 강조
- 지도는 미니멀한 스타일(Greyscale 또는 Light)을 사용하여 마커가 돋보이게 처리
- 2030 유저들이 공유하고 싶어지는 깔끔한 레이아웃
