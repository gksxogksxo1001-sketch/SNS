# [UI Spec] 인증 (로그인 및 회원가입)

## 1. 개요
사용자가 서비스를 시작하기 위해 거치는 첫 관문으로, 쉽고 빠른 인증 경험을 제공하는 시스템입니다.

## 2. 화면 구성 요소

### 2.1 로그인 메인 화면 (Login Main)
- **Top**: 앱 로고 및 환영 문구 ("여행의 순간을 기록해보세요")
- **Center**: 
    - 이메일 입력창 (Email)
    - 비밀번호 입력창 (Password)
    - [로그인] 버튼 (강조색)
- **Bottom**:
    - [아이디/비밀번호 찾기] 링크
    - [간편 회원가입] 링크
- **SNS Login Section**:
    - "다른 계정으로 로그인" 구분선
    - 카카오, 네이버, 구글 아이콘 버튼 (가로 배열)

### 2.2 이메일 회원가입 화면 (Signup)
- **Header**: 뒤로가기 버튼, [회원가입] 타이틀
- **Fields**:
    - 이메일 (중복 확인 버튼 포함)
    - 비밀번호 (8자 이상, 영문/숫자 혼합 가이드)
    - 비밀번호 확인
    - 본인 확인 
    - 닉네임 (중복 확인)
- **Footer**: [가입 완료] 버튼

## 3. 주요 인터랙션
- **입력 검증**: 이메일 형식이 맞지 않거나 비밀번호가 불일치할 경우 입력창 하단에 붉은색 에러 메시지 즉시 노출
- **로그인 성공**: 메인 피드 화면으로 즉시 이동
- **SNS 로그인 클릭**: 해당 플랫폼의 인증 팝업/페이지 노출

## 4. 디자인 포인트
- 화이트 배경에 깔끔한 입력창 스타일 (Border-radius 적용)
- 브랜드 컬러를 로그인 버튼에 적용하여 주목도 향상
- 모바일 가상 키보드가 올라올 때 입력창이 가려지지 않도록 스크롤링 처리
