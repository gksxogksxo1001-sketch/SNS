# [UI Spec] 정산 대시보드 및 더치페이 계산기

## 1. 개요
그룹 여행 중 발생한 모든 지출을 취합하고, 멤버별 분담금을 계산하여 투명하게 정산하는 화면입니다.

## 2. 화면 구성 요소

### 2.1 정산 메인 (Settlement Dashboard)
- **상단 요약 섹션**:
    - [총 지출액]: 그룹 전체 총 합계
    - [나의 미정산 금액]: 본인이 지불해야 할 남은 금액
- **지출 타임라인**: 
    - 날짜별/항목별 지출 리스트
    - 항목 클릭 시 해당 게시물(연관된 경우)로 이동 가능
    - [결제자], [카테고리], [총액] 표시 및 [정산 완료] 체크 상태 노출
- **하단 액션**: [새 지출 추가] 및 [정산하기] 버튼

### 2.2 더치페이 계산기 (Dutch Pay Calculator)
- **지출 정보**: 카테고리, 날짜, 장소, 총 결제 금액 입력
- **분담 방식 설정**:
    - **1/N 방식**: 참여 인원 모두에게 균등하게 분배
    - **직접 입력**: 특정 인원의 금액을 수동으로 입력 (예: 술을 안 마신 사람 제외 등)
- **결과 미리보기**: "A님이 B님에게 00,000원 송금 전" 스타일의 직관적인 결과 요약

## 3. 주요 인터랙션
- **송금 요청 알림**: [정산하기] 클릭 시 미결제 멤버들에게 앱 내 알림 및 푸시 전송.
- **실시간 상태 공유**: 한 명이라도 입금을 확인하고 상태를 변경하면 그룹 내 모든 멤버에게 즉시 반영.

## 4. 디자인 포인트
- 숫자의 가독성을 최우선으로 하며, 브랜드 컬러(Teal)를 적극 활용하여 신뢰감 부여.
- 입금 확인 시 체크박스 애니메이션으로 명확한 피드백 제공.
