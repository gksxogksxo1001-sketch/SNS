# [UI Spec] 게시물 및 장소 태그 작성

## 1. 개요
여행 중 사진과 텍스트를 기록하고, 태그를 통해 위치와 비용 정보를 간편하게 등록하는 화면입니다.

## 2. 화면 구성 요소

### 상단 영역 (Header)
- **좌측**: [X] 취소 버튼
- **중앙**: [새 게시물] 타이틀
- **우측**: [등록] 버튼 (활성화 시 브랜드 컬러)

### 이미지 선택 영역
- 가로 스와이프 가능한 이미지 미리보기
- [사진 추가] 아이콘 버튼 (최대 10장)

### 텍스트 입력 영역
- **본문**: "이번 여행은 어떠셨나요?" (Placeholder)
- **태그 안내**: "#장소 @멤버 입력 시 자동 인식됩니다."

### 부가 정보 입력 (Price & Visibility)
- **비용 입력 섹션**: 
    - [항공] [숙소] [식비] [기타] 아이콘 버튼
    - 클릭 시 금액 입력 창 활성화
- **공개 범위**: [전체] [친구] [그룹] 선택 라디오 버튼/토글

## 3. 주요 인터랙션
- **태그 자동 완성**: 본문에 `#` 입력 시 근처 장소 추천 리스트 팝업, `@` 입력 시 그룹 멤버 리스트 팝업
- **이미지 편집**: 등록 전 사진 순서 변경(드래그 앤 드롭) 및 간단한 필터 적용
- **비용 요약**: 입력된 비용들의 총합이 실시간으로 하단에 노출

## 4. 디자인 포인트
- 키보드가 올라온 상태에서도 주요 입력 항목이 한눈에 들어오는 레이아웃
- 장소 태그가 본문 내에서 파란색으로 강조되어 보이게 처리
- 비용 입력은 복잡한 폼 대신 직관적인 아이콘-금액 방식 사용
