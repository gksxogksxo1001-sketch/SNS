# SNS 앱 화면 목록 및 정보 구조도 (IA)

## 1. 메인 메뉴 구조
1. **홈 (피드)**: 친구 및 그룹의 게시물 탐색
2. **지도 (경로)**: 내 여행 및 그룹 여행 경로 시각화 탐색
3. **작성 (+)**: 게시물 및 장소 태그 작성
4. **알림**: 좋아요, 댓글, 멘션, 정산 알림
5. **프로필 (마이페이지)**: 내 정보, 여행 아카이브, 친구 관리

## 2. 화면별 상세 리스트

| Depth 1 | Depth 2 | 주요 기능 |
| :--- | :--- | :--- |
| **인증** | 스플래시 / 가이드 | 서비스 소개 및 앱 시작 |
| | 로그인 / 회원가입 | 이메일 및 소셜 로그인 |
| | 프로필 초기 설정 | 여행 성향 설정 (#태그) |
| **홈(피드)** | 게시물 리스트 | 사진, 텍스트, 위치태그, 공유/좋아요/댓글 |
| | 댓글 상세 | 댓글 목록 및 멘션(@) 작성 |
| **여행 기록** | 게시물 작성 | 사진 선택, 텍스트 입력, #/@ 위치 태깅, 비용 입력 |
| | 그룹 생성/관리 | 멤버 초대, 그룹 기본 정보 설정 |
| **지도/정산** | 경로 탐색 지도 | 등록된 장소들 연결 경로 표시, 효율 분석 가이드 |
| | 비용 정산판 | 개별 지출 내역 합산 및 더치페이 결과 표시 |
| **프로필** | 내 프로필 홈 | 나만의 여행 지도(핀), 여행 통계, 게시물 모아보기 |
| | 설정 | 알림 설정, 계정 관리 |
