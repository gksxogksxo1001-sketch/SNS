# [UI Spec] 메인 피드 및 여행 게시물 화면

## 1. 개요
사용자가 팔로우한 친구나 소속된 그룹의 여행 게시물을 탐색하는 앱의 메인 화면입니다.

## 2. 화면 구성 요소 (Wireframe Layout)

### 상단 영역 (Header)
- **좌측**: 서비스 로고 (텍스트 또는 아이콘)
- **우측**: 검색 아이콘, 알림 아이콘 (새 알림 시 레드 닷 표시)

### 중앙 영역 (Content Feed)
- **게시물 카드 (Post Card)**:
    - **Header**: 프로필 이미지, 닉네임, 여행 그룹 이름(있을 경우), 작성 시간
    - **Body**: 
        - 이미지/동영상 슬라이드 (가로 스와이프)
        - 장소 태그 칩 (클릭 시 지도로 이동)
        - 비용 요약 바 (예: "✈️ 35만원 | 🏨 20만원")
        - 본문 텍스트 (더보기 기능 포함)
    - **Footer**: 
        - 좋아요 아이콘 (카운트), 댓글 아이콘 (카운트), 공유하기 아이콘
        - 베스트 댓글 1개 노출

### 하단 영역 (Navigation Bar)
- 피드(Active), 지도, 작성(+), 알림, 프로필 아이콘 구성

## 3. 주요 인터랙션
- **장소 태그 클릭**: 해당 장소가 중앙에 위치한 [지도 경로 화면]으로 즉시 이동
- **비용 요약 바 클릭**: 해당 게시물과 연결된 상세 비용 정산 내역 팝업/화면 노출
- **@태그 클릭**: 태그된 사용자의 프로필로 이동
- **새로고침**: 피드 상단에서 아래로 당기기 (Pull to refresh)

## 4. 디자인 포인트
- 카드 간 간격을 여유 있게 두어 시각적 피로도 감소
- 이미지 위주의 레이아웃으로 여행의 생생함 강조
- 폰트는 가독성이 좋은 샌드세리프(San-serif) 계열 사용
