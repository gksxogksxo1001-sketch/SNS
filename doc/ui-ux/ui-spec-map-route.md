# [UI Spec] 지도 기반 경로 및 효율 분석

## 1. 개요
여행 게시물의 위치 정보를 활용하여 지도 위에 동선을 시각화하고, 이동 경로의 효율성을 분석하여 가이드하는 화면입니다.

## 2. 화면 구성 요소

### 2.1 여행 지도 (Travel Map)
- **전체 화면 지도**: Mapbox/Google Maps 기반 웹 맵.
- **마커(Marker)**: 게시물이 작성된 장소에 방문 순번이 표기된 핀 노출.
- **경로 선(Polyline)**: 장소 간의 이동 순서를 실시간으로 연결하는 선.
- **장소 정보 카드**: 지도 하단에 가로로 배열된 스와이프 가능한 카드 (사진, 장소명, 태그 요약).

### 2.2 경로 효율 가이드 (Route Efficiency Guide)
- **분석 대시보드**: 전체 이동 거리와 예상 소요 시간을 분석하여 상단에 요약 표시.
- **지능형 가이드**: "A지점과 B지점 사이의 동선이 겹칩니다. 순서를 변경하여 효율을 높이세요." 등의 최적화 제안 메시지.
- **순서 편집**: 리스트에서 드래그 앤 드롭으로 장소 순서를 변경하는 편집 모드 제공.

## 3. 주요 인터랙션
- **카드-지도 동기화**: 하단 카드를 넘기면 지도의 카메라가 해당 핀으로 부드럽게 이동.
- **실시간 경로 재계산**: 순서 변경 시 즉시 지도 위의 폴리라인이 갱신되어 최단 거리 시각화.

## 4. 디자인 포인트
- 복잡한 지도 위에서 경로 선이 돋보이도록 포인트 컬러(#F4A261) 활용.
- 효율 분석 메시지는 '알고 계셨나요?' 스타일의 툴팁 또는 배너로 친절하게 제공.
