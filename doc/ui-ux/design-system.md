# 디자인 시스템 및 룩앤필 (Design System)

## 1. 브랜드 컨셉
- **키워드**: 자유, 기록, 연결, 트렌디
- **분위기**: 여행의 설렘을 담은 밝고 깨끗한 이미지 + 2030 감성의 미니멀리즘

## 2. 컬러 팔레트 (Color Palette)
- **Primary (주요색)**: `#2A9D8F` (청량한 바다/자연을 연상시키는 Teal 컬러)
- **Secondary (보조색)**: `#E9C46A` (따뜻한 햇살을 담은 Sand Yellow)
- **Point (포인트)**: `#F4A261` (일몰의 느낌을 담은 Soft Orange)
- **Background**: `#FFFFFF` (기본), `#F8F9FA` (연한 회색 배경)
- **Text**: `#212529` (Main), `#6C757D` (Sub)

## 3. 타이포그래피 (Typography)
- **폰트**: Pretendard (가독성이 뛰어나고 현대적인 샌드세리프)
- **스타일**:
    - **H1 (Title)**: 24px, Bold
    - **H2 (Subtitle)**: 18px, Semi-Bold
    - **Body**: 16px, Regular
    - **Caption**: 12px, Regular

## 4. 주요 컴포넌트 스타일
- **Buttons**: `border-radius: 12px` (부드러운 곡선), Primary 컬러 배경
- **Input Fields**: 테두리 없이 배경색(`#F1F3F5`)으로 구분하는 깔끔한 스타일
- **Cards**: `box-shadow: 0 4px 6px rgba(0,0,0,0.05)` (은은한 그림자)
- **Tag Chips**: 둥근 형태의 칩 스타일, 해시태그는 텍스트 컬러 강조

## 5. 아이콘 스타일
- 라인 기반의 얇고 세련된 아이콘 셋 활용 (예: Lucide React, Heroicons)
- 여행 관련 아이콘(비행기, 지도, 카메라 등)은 포인트 컬러 활용 가능
