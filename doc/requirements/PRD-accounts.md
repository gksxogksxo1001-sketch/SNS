# [PRD] 다중 계정 관리 및 전환 (Account Management)

## 1. 개요
여러 계정을 사용하는 여행자들을 위해 최근 로그인한 계정들을 저장하고, 원클릭으로 간편하게 전환하거나 관리하는 기능입니다.

## 2. 상세 기능
- **계정 자동 저장**: 성공적으로 로그인한 계정 정보를 로컬 스토리지에 자동 보관 (`accountManager.saveAccount`).
- **계정 전환(Switcher)**: 프로필 설정 메뉴에서 다른 계정 리스트를 확인하고 즉시 전환 (로그아웃 및 로그인 폼 연동).
- **계정 관리**: 저장된 계정 리스트에서 특정 계정을 삭제하거나 정보를 수정할 수 있는 UI 제공.
- **다중 도메인 지원**: 개인 계정, 비즈니스 계정, 그룹 관리용 계정 등의 유연한 전환 지원.

## 3. 사용자 여정 (User Flow)
1. 최초 로그인 후 세션 정보가 `localStorage`에 '최근 계정'으로 등록됨.
2. 프로필 -> 설정 -> '계정 관리' 메뉴 클릭.
3. 등록된 다른 계정 선택 시 전환 시도.
4. 기록 삭제(Trash 아이콘) 클릭 시 해당 정보 소멸.
