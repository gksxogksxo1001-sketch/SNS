# 프로젝트 개요: 2030 여행자 전용 SNS 커뮤니티

## 1. 프로젝트 목적
본 프로젝트는 20대와 30대 여행자들을 위해 여행의 생생한 순간을 쉽고 빠르게 기록하고, 다른 사람들과 효율적인 여행 정보(기능적 경로, 정산 내역 등)를 공유할 수 있는 소셜 네트워킹 서비스(SNS)를 구축하는 것을 목적으로 합니다.

## 2. 핵심 가치 (Core Values)
- **편의성**: 복잡한 입력 없이 태그(#, @)만으로 위치를 등록하고 게시물을 작성할 수 있는 환경 제공
- **협업**: 여행 전 그룹을 형성하고, 멤버들이 공동으로 게시물을 관리하며 여행을 기록하는 경험 제공
- **투명성**: 여행 비용을 명확하게 입력하고, 1인당 정산 금액을 자동으로 계산하여 비용 갈등 해소
- **효율성**: 지도 기반의 이동 경로 시각화를 통해 여행 동선의 최적화 상태를 한눈에 파악

## 3. 타겟 사용자 (Target Audience)
- **핵심 타겟**: 여행을 즐기는 20~30대 청년층 (취준생, 사회초년생 등)
- **사용자 니즈**: 
    - 자신의 여행 경험을 예쁘고 간편하게 기록하고 싶어 함
    - 친구들과 함께 여행 일정을 관리하고 비용을 공평하게 나누고 싶어 함
    - 다른 여행자의 실제 여행 경로와 비용 정보를 참고하여 합리적인 여행 계획을 세우고 싶어 함

## 4. 기대 효과
- 여행 중 발생하는 불필요한 기록의 번거로움을 줄이고 여행 자체에 집중할 수 있게 함
- 공동 관리와 자동 정산 기능을 통해 동행자 간의 신뢰와 편의성을 높임
- 검증된 유저들의 여행 데이터를 기반으로 양질의 여행 커뮤니티 생태계 조성
