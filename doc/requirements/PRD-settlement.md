# [PRD] 스마트 여행 비용 정산 (Settlement)

## 1. 개요
여행 그룹 멤버들 간의 지출 내역을 투명하게 관리하고, 자동으로 N/1 계산 및 정산 요청 알림을 발송하는 기능입니다.

## 2. 상세 기능
- **지출 자동 집계**: 그룹 멤버들이 올린 게시물의 지출 필드(총액)를 합산하여 총 여행 비용 산출.
- **정산 알고리즘**: 복잡한 지출 관계(누가 누구에게 얼마를 줄 것인가)를 최소화된 거래 횟수로 자동 계산 (`settlementService.calculateGroupSettlement`).
- **정산 현황 대시보드**: 개인별 '보낼 돈'과 '받을 돈'을 시각화하여 제공 (`SettlementListPage`).
- **송금 요청 알림**: 정산이 필요한 대상에게 클릭 한 번으로 알림 발송 (`notificationService.sendSettlementNotifications`).

## 3. 사용자 여정 (User Flow)
1. 게시물 작성 시 '그룹'과 '지출 금액' 입력.
2. '정산' 탭 이동 시 그룹별 정산 요약 확인.
3. 정산 상세 진입 시 멤버별 송금/요청 버튼 활성화.
4. 요청 버튼 클릭 시 상대방에게 푸시 알림 전송.
