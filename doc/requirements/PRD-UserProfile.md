# [PRD] 사용자 계정 및 프로필

## 1. 개요 및 목적
사용자가 자신의 여행 정체성(Identity)을 설정하고, 취향이 비슷한 다른 여행자들과 연결될 수 있도록 돕는 개별 프로필 기능을 제공합니다.

## 2. 타겟 사용자 프로파일
- 자신의 여행 스타일을 표현하고 기록하고 싶은 2030 여행자
- 비슷한 여행 취향을 가진 사람의 정보를 신뢰하는 사용자

## 3. 핵심 사용자 여정 (User Flow)
1. **마이페이지/프로필 편집** 진입
2. **프로필 사진 및 닉네임** 설정
3. 자신의 **여행 스타일 태그 선택** (예: #프로배낭러, #호캉스족, #가성비추구 등)
4. 지금까지 다녀온 **여행 국가/도시 배지** 확인
5. 팔로워/팔로잉 목록 관리

## 4. 상세 기능 명세

| 기능명 | 기능 설명 | 우선순위 |
| :--- | :--- | :---: |
| 기본 프로필 관리 | 닉네임, 프로필 사진, 한 줄 소개 설정 | High |
| 여행 스타일 태그 | 개인별 여행 취향(예산, 활동성, 숙소 선호도 등)을 태그로 설정 | High |
| 여행 통계/디스플레이 | 총 여행 횟수, 방문한 도시 수, 총 주행 거리 등을 시각화 | Medium |
| 소셜 관계 관리 | 다른 사용자 팔로우/언팔로우 및 친구 목록 관리 | Medium |
| 여행 지도 아카이브 | 내가 작성한 게시물들의 위치를 통합하여 보여주는 개인 여행 지도 | High |

## 5. 비기능적 요구사항
- **개인정보 보호**: 이메일 등 민감 정보는 프로필에 노출되지 않도록 처리
- **시각화 성능**: 여행 지도 아카이브의 핀(Pin)이 많아질 경우 클러스터링을 통한 성능 최적화
- **UX**: 2030 감성에 맞는 깔끔하고 트렌디한 프로필 레이아웃 제공
