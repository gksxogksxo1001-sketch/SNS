# [PRD] 지도 기반 경로 및 비용 정산

## 1. 개요 및 목적
그룹 멤버들이 여행 경로의 효율성을 확인하고, 발생한 비용을 투명하게 정산하여 금전적 갈등 없는 여행을 지원하는 핵심 기능입니다.

## 2. 타켓 사용자 프로파일
- 친구들과 함께 여행을 다니며 비용을 분담해야 하는 2030 그룹 여행객
- 여행 동선을 최적화하여 시간을 아끼고자 하는 효율형 여행자

## 3. 핵심 사용자 여정 (User Flow)
1. **그룹 생성** 및 멤버 초대
2. 멤버들이 올린 게시물(위치 정보 포함)이 **지도 위에 경로로 연결**됨
3. 시스템이 제공하는 **경로 효율성 가이드** 확인
4. 개별 지출 내역 입력 (게시물 작성 시 또는 별도 입력)
5. **정산하기** 버튼 클릭 시 1인당 분담금 자동 계산
6. 정산 완료 상태 체크 및 그룹 내 알림 전송

## 4. 상세 기능 명세

| 기능명 | 기능 설명 | 우선순위 |
| :--- | :--- | :---: |
| 그룹 공동 관리 | 그룹 내 누구나 게시물을 수정/삭제할 수 있는 권한 부여 | High |
| 경로 시각화 | 게시물에 등록된 위치들을 연결하여 지도 위에 이동 동선 표시 | High |
| 경로 효율 분석 | 등록된 장소 간 거리/시간을 분석하여 경로가 비효율적일 경우 안내 | Medium |
| 더치페이 자동 계산 | 총 지출을 인원수대로 나누거나 특정 비율로 정산하는 자동 계산기 | High |
| 정산 상태 대시보드 | 미정산 포스트 확인 및 정산 완료 여부 실시간 공유 | High |

## 5. 비기능적 요구사항
- **신뢰성**: 정산 금액 계산 시 소수점 처리 등 금융 데이터와 동일한 정확성 유지
- **동기화**: 그룹 멤버가 데이터를 수정할 경우 모든 멤버의 지도와 정산 정보에 즉시 반영 (Real-time)
- **보안**: 그룹 내의 금전 정보 및 사적인 여행 데이터가 외부로 유출되지 않도록 권한 관리 철저
