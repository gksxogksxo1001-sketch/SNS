# UI/UX 디자인 워킹플로우 (UI/UX Design Workflow)

이 워크플로우는 PRD를 바탕으로 사용자 친화적이고 심미적으로 우수한 앱 화면을 설계하는 과정을 안내합니다.

## 진행 단계 (Step-by-Step Process)

### 1단계: 정보 구조도(IA) 및 화면 목록 정의
앱의 메뉴 구조와 전체적인 흐름을 정리합니다.
- **활동**: 전체 메뉴 트리와 필요한 모든 화면 리스트업
- **출력물**: `doc/ui-ux/screen-list.md`

### 2단계: 핵심 화면 상세 설계 (UI Spec)
주요 화면의 요소와 인터랙션을 정의합니다.
- **활동**: 와이어프레임 수준의 레이아웃 설계, 구성 요소 배치, 버튼 클릭 시 동작 정의
- **출력물**: `doc/ui-ux/ui-spec-[screen-name].md`

### 3단계: 디자인 시스템 및 룩앤필(Look & Feel) 제안
앱의 전체적인 분위기를 결정합니다.
- **활동**: 컬러 팔레트, 폰트, 주요 컴포넌트(버튼, 입력창 등)의 스타일 정의
- **출력물**: `doc/ui-ux/design-system.md` (또는 UI Spec 내 포함)

## 출력 파일 경로 규칙
- 화면 목록 및 IA: `doc/ui-ux/screen-list.md`
- 화면 상세 설계서: `doc/ui-ux/ui-spec-[screen-name].md`
- 디자인 시스템: `doc/ui-ux/design-system.md`
